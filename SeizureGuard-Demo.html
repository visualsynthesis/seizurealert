How It Works

From detection to alert in under 15 seconds. The wearer can always cancel with a single tap.

1

Monitor

Accelerometer reads motion at 50 Hz on a background thread. Zero battery drain from the UI.

2

Detect

Sustained high-magnitude movement for 5+ seconds triggers a seizure-like motion pattern match.

3

Countdown

10-second countdown with haptics every second. The wearer can tap "I'M OK" to cancel instantly.

4

Alert

Critical notification fires immediately. SMS opens pre-filled with GPS location and Google Maps link.

Key Features

Built for reliability when it matters most.

💨

50 Hz Motion Analysis

Accelerometer data processed on a dedicated background queue. No UI lag, no missed samples.

🚨

Critical Notifications

Fires a critical-priority local notification immediately with no user interaction required.

📍

Live GPS Location

Captures real-time coordinates and includes a Google Maps link in every alert.

🔄

Auto-Restart

Monitoring automatically resumes after an alert or cancellation. No manual restart needed.

♿

Full Accessibility

VoiceOver labels and hints on every interactive element. Tested for screen reader compatibility.

🔒

Privacy First

All data stays on-device. No cloud, no accounts. Includes Apple's required privacy manifest.
